# Сайт для заказа кроссовок. 


 ## Ссылка на демо: 
  https://kiki-do.github.io/react-sneakers/

 ## Стек технологии: 
 - HTML 
 - SASS 
 - React 
 - JavaScript
 - Axios

  
 
  

 ## Реализовано:
- Возможность добавлять в корзину и в закладки кроссовки 
- Поиск по названию   
- Изменение количества товара в корзине
- Удаление кроссовок с корзины
